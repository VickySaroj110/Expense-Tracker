# Daily Expense Tracker

A web-based **Expense Tracker** built with PHP and MySQL to help users manage daily expenses, set monthly goals, view reports, and track spending through a clean dashboard.

## Features

- User registration and login
- Dashboard with expense summary
- Add, edit, and manage expenses
- Set monthly spending goals
- Expense reports and analytics
- Profile management
- Logout system
- Responsive UI with Bootstrap
- Theme toggle for light/dark mode

## Tech Stack

- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Tools:** XAMPP / WAMP / LAMP

## Project Structure

- `config.php` - Database connection and session helper
- `login.php` - User login page
- `register.php` - User registration page
- `dashboard.php` - Main dashboard
- `add_expense.php` - Add new expense
- `manage_expenses.php` - View and manage expenses
- `editexpense.php` - Edit expense
- `set_goal.php` - Set monthly goal
- `reports.php` - Expense reports
- `profile.php` - User profile
- `logout.php` - Logout and session destroy
